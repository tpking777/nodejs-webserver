# Primer Web Server en NodeJS

Verificar la version de NodeJS y NPM
node -v && npm -v


`$ npm install marked`



## Project setup

### Node.js Server
```
cd node-js-server
node server.js
```

### Angular Client
```
cd angular-12-client
```
Run `ng serve --port 8081` for a dev server. Navigate to `http://localhost:8081/`.
